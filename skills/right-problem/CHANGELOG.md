### Changed

- Added material-ambiguity and observable-success validation guidance.

- Clarified that checklists, Five Whys chains, and resource tests are heuristics rather than proof.

### Changed

### Added

- Initial Golden Skill reference package.
- Platform-independent metadata and schema-conforming skill definition.
- Workflow, validation, conversation, evaluation, and rubric contracts.
- Simple, business, learning, and failure examples.
- Fifteen conformance test cases and normative references.
