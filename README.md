# ThinkingOS

ThinkingOS is an open-source foundation for organizing reusable reasoning workflows, knowledge, prompts, schemas, adapters, and examples.

> **Project status:** Initial architecture. No skills are implemented yet.

## Project structure

| Path | Purpose |
| --- | --- |
| `docs/` | Documentation and design notes |
| `core/` | Core runtime and domain abstractions |
| `skills/` | Skill packages (intentionally empty) |
| `knowledge/` | Curated knowledge assets |
| `schemas/` | Data and configuration schemas |
| `examples/` | Usage examples |
| `tests/` | Automated tests |
| `prompts/` | Prompt assets and templates |
| `adapters/` | Integration adapters |
| `.github/` | GitHub community and automation configuration |

## Getting started

ThinkingOS is currently at the architecture stage. Implementation guidance will be added as the project evolves.
