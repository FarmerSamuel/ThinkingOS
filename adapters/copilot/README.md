# GitHub Copilot Adapter

GitHub Copilot is treated as an MCP host. The adapter preserves ThinkingOS
state and structured output through MCP while Copilot owns model choice,
permissions, and confirmation UI.

See GitHub's official guide to [extending Copilot Chat with MCP](https://docs.github.com/en/copilot/how-tos/provide-context/use-mcp-in-your-ide/extend-copilot-chat-with-mcp).
