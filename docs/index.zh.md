# ThinkingOS

ThinkingOS 是開源且不綁定 AI 平台的思考框架，透過可重複使用的思考技能提升人類推理品質。它不是提示詞集合，也不依賴特定模型、供應商、使用者介面或工具協定。

## 框架提供的能力

- 通用的推理角色與核心原則。
- 所有技能共用、以驗證為先的執行引擎。
- 可由機器驗證的技能、評估、輸出與語言 Schema。
- 具明確相依關係與轉換條件的有向思考圖譜。
- 十三個具有版本化契約與測試的思考技能。
- 與技能邏輯分離、可重複使用的知識層。
- 不改變框架語意的平台 Adapter。

## ThinkingOS 如何運作

```mermaid
flowchart LR
    I["輸入"] --> V["驗證"]
    V --> E["評估思考元素"]
    E --> L["執行邏輯"]
    L --> Q["評估推理"]
    Q --> O["結構化輸出"]
    O --> N["下一個技能"]
```

當出現關鍵歧義、矛盾或缺少相依條件時，引擎會停止或回到前一階段。流暢的文字永遠不會被當成推理有效的證據。

## 專案狀態

ThinkingOS v1.0 是第一個穩定版。公開 SDK API 與已發布契約遵循語意化版本；供應商模型的可用性不屬於相容性保證。
