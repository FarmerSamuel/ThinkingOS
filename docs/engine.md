# Reasoning Engine

The normative ThinkingOS Engine specification is maintained in [`core/engine.md`](https://github.com/FarmerSamuel/ThinkingOS/blob/main/core/engine.md).

This documentation route exists so the Engine appears in the MkDocs navigation without duplicating its versioned contract.

## Pipeline summary

```text
Collect Inputs
↓
Validate Inputs
↓
Detect Missing Information
↓
Evaluate Thinking Elements
↓
Run Logic Rules
↓
Evaluate Reasoning
↓
Generate Evaluation
↓
Generate Output
↓
Recommend Next Skill
```

Read the [normative Engine specification](https://github.com/FarmerSamuel/ThinkingOS/blob/main/core/engine.md) for stage responsibilities and exit conditions.
