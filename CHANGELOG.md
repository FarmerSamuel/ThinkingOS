# Changelog

All notable changes to this project are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and versions follow
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Complete Traditional Chinese documentation with one-to-one page parity,
  localized navigation, search, and an English/繁體中文 language switcher.

## [1.0.0] - 2026-07-15

### Added

- Universal ThinkingOS Core Framework and validation-first reasoning Engine.
- Official Skill, evaluation, output, language, and adapter specifications.
- Thirteen production-ready, composable Thinking Skills and directed Registry.
- Reusable critical-thinking, systems-thinking, problem-solving, and
  decision-making knowledge base.
- Python SDK for Skill discovery, schema validation, graph traversal, and state.
- OpenAI, Claude, Gemini, Cursor, GitHub Copilot, and MCP adapters.
- MkDocs documentation site, community templates, CI, Pages, and release flows.

### Security

- Fail-closed adapter parsing, explicit secret boundaries, dependency updates,
  security policy, and human-in-the-loop MCP guidance.

## [0.1.0] - 2026-07-15

### Added

- Initial open-source project architecture and community documentation.
